Приложение состоит из двух частей.

Клиентская часть позволяет отправить выбранное изображение на сервер.

Серверная часть автоматически отображает полученное изображение (без сохранения на диск).

Баги:

1) После успешной отправки изображения на сервер, на стороне клиента не получается выбрать другое изображение для отправки (из уже открытого диалогового окна выбора файла). Приходится отключаться.

2) Большие изображения могут отображаться неполностью (на стороне сервера).
